# Claude Code 設定ディレクトリ

このディレクトリには、Claude Codeが常に参照するプロジェクト設定ファイルが含まれています。

## 📁 ファイル一覧

### `project-context.md`
プロジェクト全体のコンテキスト情報を記載した設定ファイル。

**主な内容:**
- プロジェクト概要と技術スタック
- 環境構成（Production/Staging）
- ディレクトリ構造
- 重要な環境変数
- データベーススキーマ
- 重要な実装パターン（高度な決済フロー、Stripe 2段階決済など）
- よく使うコマンド
- コーディング規約
- 注意事項
- 主要ドキュメントへのリンク

## 🎯 使い方

Claude Codeは、このディレクトリ内のファイルを自動的に読み込み、プロジェクトのコンテキストとして参照します。

### 新しいClaude Codeセッション開始時
1. Claude Codeが`.claude/project-context.md`を自動的に読み込む
2. プロジェクトの技術スタック、環境構成、コーディング規約などを把握
3. 適切なコード提案やドキュメント参照が可能になる

### 設定ファイルの更新
プロジェクトに変更があった場合は、このファイルも更新してください：
- 新しい技術スタックの追加
- 環境変数の変更
- 新しいドキュメントの追加
- ディレクトリ構造の変更

## 📝 カスタマイズ

プロジェクト固有の情報を追加したい場合は、`project-context.md`を編集してください。

**推奨する追加項目:**
- チーム固有のコーディング規約
- よく遭遇する問題とその解決策
- 重要なAPI仕様
- デプロイフロー
