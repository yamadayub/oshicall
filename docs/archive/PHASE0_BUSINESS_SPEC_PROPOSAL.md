# Phase 0: 業務仕様の作成/更新案

## ビジネスルール（追加/更新案）

### BR-008: 待機室での入室状態表示ルール（新規追加）

- **説明**: 待機室（CallPage）では、各参加者の実際の入室状態を正確に表示する
- **条件**: ユーザーがCallPageにアクセスした時
- **結果**:
  - 自分がCallPageに入室済み、相手が未入室 → 自分: 待機中、相手: 未入室
  - 自分がCallPageに入室済み、相手もCallPageに入室済み → 両者: 待機中
  - 自分がCallPageに入室済み、相手がDaily.coセッションに入室済み → 自分: 待機中、相手: 参加済み
  - 自分がDaily.coセッションに入室済み、相手が未入室 → 自分: 参加済み、相手: 未入室
  - 自分がDaily.coセッションに入室済み、相手もDaily.coセッションに入室済み → 両者: 参加済み

**用語定義**:
- **待機中**: CallPageに入室しているが、Daily.coセッションには未入室
- **参加済み**: Daily.coセッションに入室済み（実際の通話中）
- **未入室**: CallPageにも入室していない

### BR-009: インフルエンサーの早期入室（新規追加）

- **説明**: インフルエンサーは通話準備のため、開始時刻より前に入室できる
- **条件**: 
  - インフルエンサーのみ適用
  - 通話開始時刻の15分前以降
- **結果**: 
  - インフルエンサーは15分前からCallPageに入室可能
  - インフルエンサーは15分前からDaily.coセッションに入室可能
  - ファンは開始時刻からCallPageに入室可能
  - ファンは開始時刻からDaily.coセッションに入室可能

### BR-010: ファンの入室タイミング（新規追加）

- **説明**: ファンは通話開始時刻から入室できる
- **条件**: 
  - ファンのみ適用
  - 通話開始時刻以降
- **結果**: 
  - ファンは開始時刻からCallPageに入室可能
  - ファンは開始時刻からDaily.coセッションに入室可能
  - 開始時刻前はCallPageに入室できない

### BR-002: 通話アクセス制限（更新案）

**現状**:
- 予定時刻の前後30分のみ通話可能

**更新後**:
- インフルエンサー: 予定時刻の15分前から入室可能
- ファン: 予定時刻から入室可能
- 両者とも予定時刻の30分後まで入室可能
